JKET Parent Registration App

Django | HTML | CSS | Bootstrap

This application is made for Japan Kauri Education Trust in Auckland, New Zealand where they hold a few Japanese classes everyday. They were using a pen and paper when parents drop-off/pick-up their children. One stone two birds. I made this project as my Django capstone project and also for JKET. This simple application allows JKET admins and parents/guardians to have a central location to record drop-offs/pick-ups as well as to check logs. Even more, they can save the cost of papers! The live version is up and running.

NZD-JPY Exchange Rate Alert

Python3

Being from Japan and living in New Zealand, I need to keep an eye on currency exchange rate to transfer money. Being a mother of 3 children plus a dog, my day goes so fast and I can't keep thinking about checking today's rate. So I created this mini app. It uses crontab in my Ubuntu and sends me an email when the currency rate goes below certain price, so that I can completely forget about checking the daily rate.


TradeMe Rental Research

Python3 | Selenium

While we were deciding how much we should be renting our rampus room downstairs, I created this program that automatmatically collects certain conditions of rental property to a separate Google Form and lists up in a csv file so that I can see the information at once. By changing the search filter, I can use it for different location, or instead for purchasing property to find out average price range, the number of rooms and bathrooms.
